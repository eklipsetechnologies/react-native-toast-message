# eklipse-react-native-toast-message



Animated toast message component for React Native.

![toast gif](./docs/toast.gif)

## Features

- 🚀 Imperative API
- 📦 Very lightweight (~40 kB)
- ⌨️ Keyboard-aware
- 🎨 Customizable layouts
- 🔧 Flexible config
